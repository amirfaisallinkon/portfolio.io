My Working Fields

HTML

HTML (HyperText Markup Language) is the foundation of every web page. It structures content like text, images, and videos, making them displayable on the web.

CSS

CSS (Cascading Style Sheets) adds design and styling to web pages. It enables you to change colors, layouts, and fonts, making your web pages visually appealing.

JavaScript

JavaScript brings interactivity to web pages. It allows you to create dynamic content, handle user inputs, and implement advanced functionalities like animations.
